# GitHub Pages Privacy Policy

This folder is ready for GitHub Pages.

Suggested publish path:

- Source: `Deploy from a branch`
- Branch: your default branch
- Folder: `/docs`

After GitHub Pages is enabled, your privacy policy URL will usually be:

`https://github.com/tjshif-ops/privacy-policy/`

The actual page file is:

- `docs/privacy-policy/index.html`
